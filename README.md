# Wisdom Path Apply

A web application built with React, TypeScript, TanStack Start, and Vite.

## Tech Stack

- React 19
- TypeScript
- Vite 7
- TanStack Start / Router / Query
- Tailwind CSS 4
- Radix UI primitives
- Cloudflare Wrangler

## Getting Started

## Prerequisites

- Node.js 18+ (or Bun if preferred)

## Install

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Format

```bash
npm run format
```
